Discourse Representation Theory

This grammar is an implementation of Reinhard Musken's ideas for adding discourse representation theory (DRT) as a semantics for the Lambek calculus. It uses the simple associative Lambek calculus, but in the interest of keeping the resulting representations a bit more compact there is a strategic use of some non-associative implications not found in the original paper.

Grail permits the use of discourse representation structures (DRSs) in its semantics using the following syntax.

drs(ListOfVariables, ListOfConditions)

So a DRS like the following

x
man(x)
enter(x)
smile(x)

will be represented as

drs([X], [man(X),enter(X),smile(X)])

whereas a slightly more complicated DRS with embedded DRSs and an implication in the conditions

Remarks

  • Note that there is currently no mechanism included to compute the different possible bindings of the personal pronouns.
  • Note as well that the grammar proposes multiple equivalent readings for many sentences, since all np's in the grammar are lifted.
